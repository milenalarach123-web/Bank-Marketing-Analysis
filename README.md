# Bank Marketing Analysis
Proyecto de análisis de campañas bancarias utilizando Excel y Python.

Objetivo:
Identificar el perfil de los clientes con mayor probabilidad de suscribirse a un depósito a plazo fijo.

## Tecnologías
- Excel
- Python
- Pandas
- Matplotlib
